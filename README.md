# Портфолио - проекты и артефакты

Репозиторий содержит несколько учебных/портфолио-проектов, оформленных как набор артефактов: требования, диаграммы, схемы БД, SQL и контракт API.

## Содержание

- `salonbeauty/` - проект "Beauty Salon Network" (API, БД, BPMN, use-cases)
- `musicservice/` - аналитический мини-проект по MusicService
- `links.md` - ссылки на внешние доски и материалы

## Быстрая навигация

- `salonbeauty/api/openapi.yaml` - OpenAPI контракт
- `salonbeauty/database/` - DDL/DML/queries и ERwin модели
- `salonbeauty/bpmn/` - BPMN диаграммы
- `salonbeauty/docs/` - use-cases и требования
